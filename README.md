# DP-100 Exam Preparation (2026)

## Project Overview
This repository is dedicated to preparing for the Microsoft Azure Data Scientist Associate (DP-100) certification exam. The goal is to master the necessary skills and knowledge to pass the exam scheduled for **March 11, 2026**.

## Environment Setup
* **Account**: Azure Student Account
* **Compute Resources**: The focus is on using **Dedicated CPU Compute Clusters** (B-series/D-series) to ensure reliable performance and availability, avoiding the potential interruptions of low-priority instances.

## Repository Structure
The repository is organized into the following key directories:

```
.
├── Notebooks/      # Jupyter notebooks for interactive learning and experiments
├── Scripts/        # Python scripts for training, evaluation, and deployment
├── Data/           # Datasets used for experiments
└── Environments/   # Conda environment specifications and Dockerfiles
```

## Study Programme

### Week 1: Azure Machine Learning Workspace, Data Assets, and Compute Resources
- [ ] Create and configure the Azure Machine Learning Workspace
- [ ] Set up and manage Data Assets (Datastores and Datasets)
- [ ] Provision Compute Resources (Compute Instances and Compute Clusters)

### Week 2: Orchestrating training with Jobs and Pipelines (v2 SDK)
- [ ] Configure and run Command Jobs using the v2 SDK
- [ ] Design and execute Machine Learning Pipelines

### Upcoming
- [ ] Model training techniques
- [ ] Hyperparameter tuning
- [ ] Deploying models to Managed Endpoints
